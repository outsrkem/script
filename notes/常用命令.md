## 创建临时目录或文件

```shell
Usage: mktemp [OPTION]... [TEMPLATE]
  -d, --directory     用于创建目录而非文件
  -u, --dry-run       创建的文件会在mktemp之行结束被删除，所以并非真正创建文件，只是打印生成的文件名，后缀为随机数字
  -q, --quiet         执行时若发生错误，不会显示任何信息。
  -t                  用于在/tmp目录下创建临时文件            	
```

实例

```shell
tempfile=$(mktemp -t test-XXXXXXXXXXXX) && echo $tempfile
tempfile=$(mktemp -td test-XXXXXXXXXXXX) && echo $tempfile
```
